Did you know that you can use ChatGPT to practise for a job interview?

Practising for an interview is best done with a real human, but if you don't have a friend (or access to a teacher), ChatGPT is better than nothing!

Here is an interview I had for a waiter in a Korean restaurant: ChatGPT chat log

(As a teacher, I reckon ChatGPT's questions were pretty good! There was a lot of (difficult?) natural language that I've heard actual job interviewers use, so you have to use different language skills in order to understand what's being asked.)
